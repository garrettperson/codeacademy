CSS Selectors

The following are a few common types of CSS Selectors

Type of Selector Description
Type, Class, & ID Selectors These selectors can target an HTML element such as <h1> via h1 { }, or these selectors may target a class (ex: .box { }), or they may target an ID (ex: #unique { }).
Attribute Selectors These selectors allow one to select different elements based on the presence of a certain attribute, for ex: a[title] { } . These selectors can also select based on the presence of an attribute with a particular value, for ex: a[href="https://example.com"] { } .
Pseudo-classes Pseudo-classes style certain states of an element, for ex: a:hover { } selects an element only when it is being hovered over by the mouse pointer. .
Pseudo-elements Pseudo-elements select a certain part of an element rather than the element itself, for ex: p::first-line { } selects the first line of text of a >p> element. It acts as if a <span> was wrapped around the first formatted line and then selected.
Combinators This group of selectors combine other selectors in order to target elements within our documents, for ex: article > p { } selects paragraphs that are direct children of <article> elements using the child combinator (>).
